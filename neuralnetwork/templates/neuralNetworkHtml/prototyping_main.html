Neural Network

Software Prototyping -- Neural Network for Diabetes Prediction


Description：


1st Step, we can do the data processing for getting a better neural network model;
2nd Step, we can use the exist training data to train the neural networks MLFF_BP, SVM, and test the neural networks which are trained already, then we can get the best neural network for the situation;
3rd Step, we can use the better neural network for the situation to predict diabetes for the person who give the related parameters.

Notice：


1. The raw data is there and cannot change; algorithm of data processing is mostly based on the Box Plot and the domain knowledge;
2. The training data and the test data will be uploaded together with the CSV file and 75% for trainning, 25% for test;
3. We can do the prediction with two options one instance or a group of instances; for the group one, we just need upload the related CSV file, and for the single one, we need gather the 8 parameters which are number of times pregnant, plasma glucose concentration a 2 hours in an oral glucose tolerance test, diastolic blood pressure (mm Hg), triceps skin fold thickness (mm), 2-hour serum insulin (mu U/ml), body mass index (weight in kg/(height in m)^2), diabetes pedigree function, age (years).
